This small program is a matrix calculator that works with 6 matrixes at a time and has the following commands:
read_mat - recieves a matix as a input, accepts at least one number, each number over 16 is ignored.
print_mat - prints an existing matrix. 
add_mat - adds two matrixes and puts the result in a third.
sub_mat - same with substraction.
mul_mat - same with multifiaction
mul_scalar - multiflies a matrix with a flout number and puts result in other matrix.
trans_mat - transform a matrix.
stop - stops the program.

The program detects errors and warns the user about them, for example:
illegal command, illegal matrix name, illegal number, etc.
